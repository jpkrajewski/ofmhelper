Before you download: if a link fails with a login/cookies error, follow these steps once:
  1. Open a browser and log into TikTok, YouTube, and Instagram (use the account you want this tool to download as).
  2. Install the "Get cookies.txt LOCALLY" extension for your browser.
  3. Click the extension icon and choose "Export All Cookies" — this saves one cookies.txt file covering all sites.
  4. Go to the Upload Cookies page and upload that file.

Cookies expire every so often (days to weeks depending on the platform). If downloads suddenly start failing again, just repeat these steps and re-upload a fresh cookies.txt.

Paste one URL per line — Instagram posts, Twitter/X posts, Pinterest, TikTok photo posts, etc.
