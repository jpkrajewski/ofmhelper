Before you generate: this tool uses your kie.ai API key to run the job. If you don't have one yet, follow these steps once:
  1. Go to kie.ai and create an account (or log in if you already have one).
  2. Top up credits on your account — generations are billed per job, so you'll need balance before running anything.
  3. Open your API key dashboard and copy your key.
  4. Paste that key into the API key field below.

Your key is only used for this job — it isn't saved or shared. If a job fails with an authorization or credit error, check your kie.ai balance and key first.
